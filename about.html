Services I Offer



Wireframe

In user interface design, wireframes are used to create layouts for websites, web apps, mobile apps, and more. Wireframes are even used in posters and infographics to help improve the user experience.

UI/UX

A UI (User Interface) deals with the application's graphical layout, which includes buttons, screen layout, animations, transitions, micro-interactions, and so on. In short, UI is all about how things look.

App Design

Mobile app design encompasses both the user interface (UI) and user experience (UX). Designers are responsible for the overall style of the app, like the colour scheme and the types of buttons and widgets the user will use.
